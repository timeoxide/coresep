<h1 align="center">
   <b>
        Installation
   </b>
</h1>

To install with npm use:

```bash
npm install coresep --save-dev
```
With pnpm:

```bash
pnpm add coresep --save-dev
```
With yarn:

```bash
yarn add coresep --dev
```
